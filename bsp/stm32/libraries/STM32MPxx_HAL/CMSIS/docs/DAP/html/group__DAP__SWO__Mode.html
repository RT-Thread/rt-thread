DAP_SWO_Mode (0x18):

Sets the SWO trace capture mode.

| BYTE | BYTE |
> 0x18 | Mode |
|******|******|
  • Mode:
    • 0 - Off (default)
    • 1 - UART
    • 2 - Manchester
