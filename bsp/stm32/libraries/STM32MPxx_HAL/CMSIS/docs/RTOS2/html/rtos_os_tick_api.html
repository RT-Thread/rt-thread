OS Tick API

The CMSIS OS Tick API may be used by an RTOS implementation to be easily potable across the different Cortex-M and Cortex-A processors. It provides a generic interface to a kernel system tick timer and defines the following functions:

  • The Reference for OS Tick API provides details about these functions:
    • OS_Tick_Setup : Setup OS Tick timer to generate periodic RTOS Kernel Ticks.
    • OS_Tick_Enable : Enable OS Tick timer interrupt.
    • OS_Tick_Disable : Disable OS Tick timer interrupt.
    • OS_Tick_AcknowledgeIRQ : Acknowledge execution of OS Tick timer interrupt.
    • OS_Tick_GetIRQn : Get OS Tick timer IRQ number.
    • OS_Tick_GetClock : Get OS Tick timer clock frequency.
    • OS_Tick_GetInterval : Get OS Tick timer interval reload value.
    • OS_Tick_GetCount : Get OS Tick timer counter value.
    • OS_Tick_GetOverflow : Get OS Tick timer overflow status.
