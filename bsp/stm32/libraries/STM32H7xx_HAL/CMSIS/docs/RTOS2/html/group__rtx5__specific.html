Description

The RTX5 kernel can be customized for different application requirements:

  • If you are depending on the lowest power consumption possible, you need to adapt the function osRtxIdleThread to send the system to sleep mode as often as possible. In addition, use the tick-less low power functions osKernelSuspend and osKernelResume to suspend the scheduler and to stop the SysTick timer.
  • If you need to specify an alternate hardware timer as the system tick timer, you need to implement the functions osRtxSysTimerSetup, osRtxSysTimerEnable, osRtxSysTimerDisable, and optionally the function osRtxSysTimerAckIRQ.
  • If you try to find a runtime error, use the function osRtxErrorNotify to debug the error.

RTX5 interfaces to the Event Recorder to provide event information which helps you to understand and analyze the operation. Refer to Event functions for more information.
